# Blahaj Clicker
A simple site where the aim is to click on a blahaj :3

## Installation
### Prerequisites
- Bun

### Installation
1. Clone and install dependencies
```sh
git clone https://github.com/oriionn/blahaj-clicker
cd blahaj-clicker
bun install
```

2. Copy the `.env.example` to make your own
```sh
cp .env.example .env
nano .env
```

3. Start the server
```sh
bun start
```
